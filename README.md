# codex-delegator-skill

Delegation utilities packaged as a Codex skill.

## What's Included

- `delegate.sh` - Delegates a task to `codex` with structured prompts + logging
- `common-roles/` - Predefined role prompts (use with `--common-role`)
- `SKILL.md` - Skill definition + usage docs
- `references/` - Expanded role and task templates
